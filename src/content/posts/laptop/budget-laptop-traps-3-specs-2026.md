## 함정 3: eMMC 저장장치 — 편도 도로 vs 고속도로


| 항목    | eMMC 5.1                | NVMe SSD           |
| ----- | ----------------------- | ------------------ |
| 순차 읽기 | 최대 400MB/s              | **3,500MB/s 이상**   |
| 순차 쓰기 | 125~200MB/s             | **3,000MB/s 이상**   |
| OS 부팅 | 기준                      | **4~10배 빠름**       |
| 동시 작업 | 단일 큐만 처리                | **64K 큐, 64K 커맨드** |
| 용량    | 32~64GB (윈도우+기본앱으로 꽉 찬) | 256GB~2TB          |
| 업그레이드 | **불가** (메인보드 납땅)        | 교체 가능              |

> eMMC는 윈도우 업데이트만으로도 용량이 부족해 수명이 급감합니다. **절대 사면 안 되는** 저장장치입니다.

<summary>Q1. eMMC인지 SSD인지 어떻게 확인하나요?</summary>

</details>

<details>
<summary>Q2. 싱글채널인지 어떻게 알 수 있나요?</summary>

CPU-Z(무료) 프로그램을 설치하면 Memory 탭에서 Channel # 항목으로 확인 가능합니다. "Dual"이면 듀얼채널, "Single"이면 싱글채널입니다.


</details>

<details>
<summary>Q3. 프리도스(FreeDOS) 노트북이 싸던데 사도 될까요?</summary>
